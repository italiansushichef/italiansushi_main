How do I download an item set?:

Use your item set in game! Clicking the download button will open the json file of the item set in a new tab.

It should look something like this:


  1. Right click anywhere on the page and select 'Save as'

  2. Navigate to your League of Legends folder, which is where you saved it when you first installed the game.
    3. The default location for Windows users is: C:\Riot Games\League of Legends\

  3. If you want the item set to be available for all champions, navigate to:
    • League of Legends > Config > Global > Recommended

    (if you can't find the Global folder, create one in the Config folder)

  4. If you want the item set to be saved for specific champion, navigate to:
    • League of Legends > Config > Champions > **CHAMPION NAME HERE** > Recommended


  5. After saving, your item set should be available in game.
