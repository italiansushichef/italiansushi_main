Why build an item set?

In the case that you don't want to use the recommended build that league provides and want a convenient way to make your custom build you want without searching through all the items, item sets are for you! If you go to the tab marked 'Build a custome item set' on the main page, you can make one on this very site!'

Why can't I upload my item set?

Remember that you need to upload your item set in json file format! Check here for more info: https://developer.riotgames.com/docs/item-sets
