Features

iTahmSets provide 5 distinct features for users. See the tutorials in the tabs for how to use.

Uploading Item Sets

Store your item set JSON files on our website. Our website will check the file format to make sure it is in the appropriate format. You can download them on any computer.

Matchup Generator

From our dataset, we parse item information from thousands of matches and use a weighted algorithm based on the lane, opponent champion, and who won the game to generate an item set with 5 blocks: Starting Items, Early Game, Mid Game, Late Game, Full Build.

Item Set Builder

Design a custom item set in-browser that can be downloaded and used in-game.

Search Tool

Browse by champion and lane to see the most popular item sets that other users have uploaded or designed. You can vote on your favorites too!

Saved Item Set Manager

View, edit, and download all item sets saved to your account. You can save up to 10 item sets to a single account.

Data Set

Using the Riot Developmental API, we collected ~11,000 ranked players from the Plat-Diamond range and ~50,000 of their unique ranked SR matches. Due to GitHub repo limits, we currently only use a smaller set of ~5000 matches to generate lane matchups. In the future this will be increased, which will drastically improve the generated matchup results.

How to Download:

Use your item set in game! Clicking the download button will open the json file of the item set in a new tab.

It should look something like this:


  1. Right click anywhere on the page and select 'Save as'

  2. Navigate to your League of Legends folder, which is where you saved it when you first installed the game.
    3. The default location for Windows users is: C:\Riot Games\League of Legends\

  3. If you want the item set to be available for all champions, navigate to:
    • League of Legends > Config > Global > Recommended

    (if you can't find the Global folder, create one in the Config folder)

  4. If you want the item set to be saved for specific champion, navigate to:
    • League of Legends > Config > Champions > champion name > Recommended


  5. After saving, your item set should be available in game.
